# react-router-express-boilerplate
A project that shows how to connect react with simple routing + express.js REST-API

Working with:
React 16
Express 4

## Setup

```
$ npm install
```

## Running the client-side(only)

```
$ npm start
```

## Build the client-side

```
$ npm run build
```

## Running the server-side

```
$ npm run start-node
```
